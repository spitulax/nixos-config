# Hyprland Configuration

NOTE:
- Monitor configuration is defined for each hosts in `/users/<user>/hosts/<host>/hyprland.nix`.

## Scripts (`./scripts/`)

- `autorun.sh`: executed on launch
- `run.sh`: launcher for common apps
